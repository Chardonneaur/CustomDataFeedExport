# Screenshots

This directory should contain the following screenshots:

1. **main-interface.png** - The main Data Feed Export interface showing the list of feeds
2. **dimension-selection.png** - The dimension selection panel with order badges and search
3. **reorder-panel.png** - The reorder panel showing selected dimensions with arrows
4. **filter-configuration.png** - The filter section with searchable dropdowns
5. **export-dialog.png** - The export dialog with date/period options

## How to Take Screenshots

1. Navigate to your Matomo installation
2. Go to **Administration > Data Feed Export**
3. Take screenshots of each feature as described above
4. Save them in this directory with the exact filenames listed

## Recommended Screenshot Settings

- **Format**: PNG
- **Width**: 800-1200px
- **Annotations**: Optional - highlight key features if needed
